Restaurant Supply Express

Relational Database Development

Project Overview

This project involved the design, implementation, and documentation of an online system for a restaurant supply delivery service, Restaurant Supply Express. Classical methodology guided the development of a relational database management system, using MySQL Server. An enhanced entity relationship diagram was constructed to parse the business use case and reflect its requirements, after which relational and database schema were created using various SQL statements based on this framework. This has helped improve the data integrity and scalability to prevent corruption and accommodate future growth. For enhanced flexibility and usability, multiple views and stored procedures were implemented to allow system operators to access information and modify the database. The database system was integrated with a fully functional web-based application found here for user access. The database tables can be queried and modified from this online system, using the associated pages linked in the banner above.

How to Set Up & Run

  1. Navigate to the code/database/ directory. Run the init_database.sql script in MySQL.
  2. Navigate to the code/ directory in the terminal. Create a python virtual environment using the following command.
    pip -m venv env
  3. Activate the environment with the provided requirements from the requirements.txt file based on os.
    python install -r requirements.txt
  4. Update the password in the secure.py file to match your MySQL database password. This file is located in the code/setup/ directory. NEVER PUSH THIS CHANGE TO THE GIT
  5. Run a python server to serve necessary files using the manage.py file and the following command.
    python manage.py runserver
  6. Go to http://127.0.0.1:8000/restaurant_supply_express/home/ to interact with the online system.

Technologies Used

A MySQL database is connected to a Django backend that allowed for database operations to be made.
To keep the user interface consistent, Bootstrap was used for the frontend design.
