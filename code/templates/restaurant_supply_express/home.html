How to Set Up & Run (Windows)

  1. Clone Repo
  2. Run \4400_Phase4\phase4\sql\init_database.sql script in MySql
  3. (Optional) Create Python Virtual Environment
  4. Install requirements from requirements.txt file
  5. Update the password in \4400_Phase4\phase4\phase4\secure.py to match MySql database password NEVER PUSH THIS CHANGE TO THE GIT
  6. Navigate to \4400_Phase4\phase4
  7. Run python manage.py runserver in aforementioned directory
  8. Go to http://127.0.0.1:8000/restaurant_supply_express/home/

Technologies Used

To keep things as simple as possible we decided to use our already existent MySql database backend. We connected that to a Django backend that made database operations simple. To keep the UI looking consistent and to make the UI easy to build we decided to use Bootstrap for the frontend design.
